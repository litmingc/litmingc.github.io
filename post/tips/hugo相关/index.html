[Tips]Hugo主题不满意的各种修改

时间： 2020-02-27   |   分类： Hugo     |   阅读： 1595 字 ~4分钟

环境：

  • hugo_extended_0.55.6_Windows-64bit
  • hugo主题：Next

表达数学公式

  • 方案一： 先上参考：
    MathJax Support
    在Hugo中使用MathJax

    修改公式显示的大小：
    方法一：修改上述参考中的style。
    方法二：使用如下代码：

    <script type="text/x-mathjax-config">
MathJax.Hub.Config({
   "HTML-CSS": { scale: 175}
});
</script>

    但是，MathJax在渲染的时候，速度有些慢。总要等很久。 又因为听说KaTeX挺快的（用了一下，还真的）。
    参考上述改法，KaTeX的配置如下：

    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/katex@0.10.0-beta/dist/katex.min.css" integrity="sha384-9tPv11A+glH/on/wEu99NVwDPwkMQESOocs/ZGXPoIiLE8MU/qkqUcZ3zzL+6DuH" crossorigin="anonymous">
<script src="https://cdn.jsdelivr.net/npm/katex@0.10.0-beta/dist/katex.min.js" integrity="sha384-U8Vrjwb8fuHMt6ewaCy8uqeUXv4oitYACKdB0VziCerzt011iQ/0TqlSlv8MReCm" crossorigin="anonymous"></script>
<script src="https://cdn.jsdelivr.net/npm/katex@0.10.0-beta/dist/contrib/auto-render.min.js" integrity="sha384-aGfk5kvhIq5x1x5YdvCp4upKZYnA8ckafviDpmWEKp4afOZEqOli7gqSnh8I6enH" crossorigin="anonymous"></script>
<script>
renderMathInElement(document.body,
{
   delimiters: [
   {left: "$$", right: "$$", display: true},
   {left: "$", right: "$", display: false},
   ],
   ignoredTags: ['script', 'noscript', 'style', 'textarea', 'pre'],
}
);
var all=document.getElementsByClassName('katex'),i;
for(i = 0; i < all.length; i += 1) {
var tmp=all[i].parentNode;
for(;tmp.nodeName=='SPAN';tmp=tmp.parentNode);
if(tmp.nodeName == 'CODE')
   tmp.className += ' hasKatex';
}
</script>
<style>
code.hasKatex {
font: inherit;
font-size: 110%;
background: inherit;
border: inherit;
color: #515151;
}
</style>

    如同参考教程12所说的一样，公式中使用_等符号会和Markdown冲突造成渲染失败，上述解决方法是在公式外加一对``解决，同时调整公式父层code标签的样式（因为原先的code的样式背景是灰的）。

  • 方案二
    上一种方案需要``把数学公式括起来，因而在使用vscode等编辑文档时非常不便于预览。幸运的是，在使用方案一的几天后，我又找到了方案二。

    先上参考3
    首先，和方案一一样引入KaTeX的代码块。因为不套用code块，也不需要调整code块的样式了，所以直接用官方默认的示例就行了：

    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/katex@0.11.1/dist/katex.min.css" integrity="sha384-zB1R0rpPzHqg7Kpt0Aljp8JPLqbXI3bhnPWROx27a9N0Ll6ZP/+DiW/UqRcLbRjq" crossorigin="anonymous">
<script defer src="https://cdn.jsdelivr.net/npm/katex@0.11.1/dist/katex.min.js" integrity="sha384-y23I5Q6l+B6vatafAwxRu/0oK/79VlbSz7Q9aiSZUvyWYIYsd+qj+o24G5ZU2zJz" crossorigin="anonymous"></script>
<script defer src="https://cdn.jsdelivr.net/npm/katex@0.11.1/dist/contrib/auto-render.min.js" integrity="sha384-kWPLUVMOks5AQFrykwIup5lo0m3iMkkHrD0uJ4H5cjeGihAutqP0yW0J6dpFiVkI" crossorigin="anonymous"></script>
<script>
   document.addEventListener("DOMContentLoaded", function() {
      renderMathInElement(document.body, {
            // ...options...
      });
   });
</script>

    然后，安装pandoc。
    最后，在MarkDown文件的front-matter部分加上markup: "pandoc"即可。
    ««««««««手工分割线««««««««
    这里还有一些问题。其中，如果配置delimiters: 选项，一定要把符号配全，pandoc似乎会把美元符号自动替换成其他符号。
    不用pandoc，用mmark也行，但官方表示mmark未来将会被移除。

增加暗黑模式

参考：https://radu-matei.com/blog/dark-mode/

  1. 新建一个html文件编写与暗黑模式相关的代码
    我选择在\layouts\partial\floater.html路径下创建文件
    在合适的位置（比如，header.html中），导入模板文件： 
    {{- partial "floater.html" . -}}
  2. 为主题添加一个固定在角落里的图标，点击图标时触发主题的更改
    添加角落的图标： 
    <style>
.darkbutton{
   position:fixed;
   top:30px;
   right:50px;
   width: 2.3em;
   height: 2.3em;
}
</style>

<script src="//at.alicdn.com/t/font_1706978_500bk6gzog3.js"></script>
<div id="togglebutton">
   <svg class="darkbutton" aria-hidden="true"><use id="dark-mode-toggle"  xlink:href="#icon-taiyang"></use></svg>
</div>

    其中<style>中position:fixed;top:30px;right:50px;使html元素固定在右上角。 图标，使用了iconfont图标库。iconfont的使用方法官网有详细的使用说明。
    当然，你可以使用其他的图标库，甚至是文字。

  3. 获取dark模式下的css文件
    使用浏览器插件调成暗黑模式后，打开开发者管理工具下载源代码-->暗模式-->css-->global-new.css。使用其他拓展时，css的路径可能不一致。
    将css文件放到static\css目录下，重命名为dark.css
    在主题模板中导入css文件： 
    <!-- 暗黑主题 -->
<link disabled id="dark-mode-theme" rel="stylesheet" href="/css/dark.css">

    默认为不生效。

  4. 添加js脚本控制css文件的效果 
    <script>
var toggle = document.getElementById("dark-mode-toggle");
var darkTheme = document.getElementById("dark-mode-theme");
var button = document.getElementById("togglebutton");
button.addEventListener("click", () => {
   if (toggle.getAttribute("xlink:href") === "#icon-yueliang1") {
      setTheme("dark");
   } else if (toggle.getAttribute("xlink:href") === "#icon-taiyang") {
      setTheme("light");
   }
});

// the default theme is light
// 持久化存储，防止转到其他页面时，模式恢复为light
var savedTheme = localStorage.getItem("dark-mode-storage") || "light";
setTheme(savedTheme);

function setTheme(mode) {
   // 持久化存储，防止转到其他页面时，模式恢复为light
   localStorage.setItem("dark-mode-storage", mode);
   if (mode === "dark") {
      darkTheme.disabled = true;
      // 设置属性值
      toggle.setAttribute("xlink:href","#icon-taiyang"); 
   } else if (mode === "light") {
      darkTheme.disabled = false;
      toggle.setAttribute("xlink:href","#icon-yueliang1"); 
   }
}
</script>

    其中，元素的class、id、以及图标内容，需要结合自己的代码做相应的更改。
